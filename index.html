Library Management System

A project by Dimpliii 🙂

✨ About the Project

A complete digital library solution that lets librarians manage books and members, while members browse and borrow books with automatic fine tracking.

🎯 Key Features

  • 📖 Add, search & manage book catalog
  • 👥 Member registration & management
  • 📤 Issue books with 14-day loan period
  • 💰 Auto-fine calculation (Rs 5/day late)
  • 📊 Live dashboard with statistics
  • 🔍 Real-time search & filtering

🧪 How to Test (for Evaluator)
